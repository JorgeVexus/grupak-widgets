# Registro de Cambios - 2026-06-18

## Resumen de la Sesión
En esta sesión se implementó una animación de trazado de contornos secuencial en la pantalla de inicio (Slide 0) y se modularizó el widget de productos interactivos para su hospedaje en GitHub y despliegue en Vercel. Adicionalmente, se implementó una arquitectura de carga autónoma (auto-loader) en JavaScript para simplificar la integración en Webflow y se configuró el mapeo para las tipografías Neue Haas Display.

## Cambios Detallados

### Componentes y Archivos Creados
- **`widgets/productos-interactivos/`**:
  - `productos-interactivos.html`: Marcado de la sección de productos interactivos de Grupack.
  - `productos-interactivos.css`: Hojas de estilos y animaciones. Se actualizaron las variables tipográficas (`--font-title` y `--font-body`) para utilizar las fuentes cargadas en Webflow (`Neuehaasdisplayblack`, `Neuehaasdisplayroman` y `Neuehaasdisplaymediu`).
  - `productos-interactivos.js`: Se actualizó para inyectar dinámicamente sus propios estilos CSS e importar el marcado HTML desde Vercel de forma automática al detectar el contenedor raíz.
- **`README.md`**: Se simplificó la documentación de inyección en Webflow, reduciéndola a un contenedor y un único tag de script.
- **`vercel.json`**: Se agregaron cabeceras CORS (`Access-Control-Allow-Origin: *`) para habilitar peticiones HTTP fetch asíncronas desde Webflow a Vercel.

### Modificaciones en Archivos Existentes
- **`hermes-preview-productos.html`**:
  - Ajuste de los SVGs de contorno para que calcen perfectamente con las resoluciones nativas de las imágenes.
  - Configuración de la animación coreográfica en `.mode-0` para el trazado escalonado en la pantalla inicial.
