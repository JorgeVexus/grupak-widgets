# Imagenes requeridas para el widget Comunidad

Coloca aqui estas cuatro imagenes en formato WebP:

- `comunidad-01-bienestar.webp`
- `comunidad-02-apoyo-social.webp`
- `comunidad-03-desarrollo.webp`
- `comunidad-04-educacion.webp`

La proporcion ideal de cada imagen es cercana a `823 x 959 px`, o mayor manteniendo esa relacion vertical. El widget usa `object-fit: cover`, asi que pueden ser mas grandes.
